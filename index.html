So your curious about functional programming? Well have a seat and listen close...
In this presentation we discuss functional programming: What it looks like, how it works, and what it's good for.
We will show examples of functional programming in conventional languages, and (most importantly) how functional programming can complement object-oriented programming.
Topics:
  • Partial Evaluation
  • Memoization
  • Currying
  • Recursion
  • Lambda Expressions
  • Reactive Programming
  • Object-oriented and Functional programming
  • Closures
